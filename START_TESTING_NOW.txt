║  ШАГ 2: БЫСТРЫЙ ТЕСТ (3 МИНУТЫ)                             ║
╚══════════════════════════════════════════════════════════════╝

   ТЕСТ A: Без 2FA
   ─────────────────
   1. Войдите с обычным пользователем
   2. Ожидается: сразу редирект на dashboard
   
   
   ТЕСТ B: С 2FA (SMS)
   ────────────────────
   1. Войдите с пользователем с 2FA
   2. Увидите диалог 2FA
   3. Выберите SMS
   4. Нажмите "Send Code"
   5. Увидите таймер: "Resend code in 60s"
   6. Введите код
   7. Нажмите "Verify Code"
   8. Ожидается: успешный вход
   
   
   ТЕСТ C: Resend Code (НОВОЕ!)
   ──────────────────────────────
   1. После отправки кода (шаг 4 выше)
   2. Дождитесь окончания таймера (60s → 0)
   3. Нажмите "Resend code"
   4. Увидите новый таймер
   5. Введите новый код
   6. Ожидается: успешный вход
