Personal calendar — testing guide

FieldWire · v1.1 · fieldwire.medicalcontractor.ca

Important: this is not the Tasks calendar. The calendar view under Tasks is disabled. The new Calendar is a separate personal calendar. Events are not linked to project tasks.

1. Two calendar modes

Global calendar Project calendar
Where Top menu Calendar Project → sidebar Calendar
Visible Personal + all projects Personal + this project
Editable Personal only Project events only
Other projects Read-only (gray) Hidden
Edit personal events in the global calendar. Edit project events in that project’s Calendar tab.

2. Event types

Type Requires my presence Behaviour
Reminder off Unlimited overlaps; no conflict checks.
Requires my presence on Warns on overlap with other presence events (all calendars).

On conflict: Save → list → Save anyway or Cancel.

3. Export (iCalendar / Google Calendar)

Green Export iCal button on both calendars:

  • iCalendar (.ics) — download for Apple Calendar, Outlook, etc.
  • Google Calendar… — downloads .ics and shows how to import in Google Calendar

Google import: + next to “Other calendars” → Import → upload the file.

Export includes all events in that view, not only the visible month.

4. Colours

Global

  • Blue — personal (editable)
  • Gray — project (read-only)
  • Amber border — requires presence

Project

  • Green — project (editable)
  • Light blue — personal (read-only)
  • Amber border — requires presence

5. Creating events

  • Month: single click selects day; double-click or + New event opens form (default 10:00–12:00).
  • Week / Day: drag a time range or double-click a slot.

6. Test scenarios

  1. Personal reminder in global calendar — create, edit, delete.
  2. Two reminders at the same time — both save.
  3. Two presence events overlapping — warning; Save anyway works.
  4. Presence in project A conflicts with presence in global calendar.
  5. Project event editable in project tab; read-only in global calendar.
  6. Event in project B not visible in project A calendar.
  7. Export iCal — download .ics; import into Google or open in Apple Calendar.

7. Limitations

  • No live sync with Google/Outlook (export/import only).
  • Events do not create or change Tasks.
  • Presence checks apply only to the logged-in user.
