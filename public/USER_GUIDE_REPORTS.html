Reports — user guide

Where in the app: Settings → Reports

Who can use it: Admin and Project Manager.

Current status: Reports exist in a basic form (daily / weekly / monthly style lists). Richer reports, exports, and custom layouts will be developed on request based on what you need for the business.

What this page is for

Saved summaries of work over a period — for example “what happened yesterday” or “this week”.

Different from Dashboard: Dashboard is a live overview; Reports are snapshots you can reopen later.

What you can do today

  • Open Settings → Reports
  • Switch type: Daily / Weekly / Monthly (if tabs are shown)
  • Pick a period from the list
  • Read the report sections that are filled in
  • Use Refresh to reload

What we will add later (on request)

  • Clearer sections and nicer layout
  • PDF / email export
  • Filters by project or team
  • Automatic scheduled reports (already started via Admin settings → Event Rules)

How to test (simple)

  1. Log in as Admin.
  2. Open Settings → Reports.
  3. If a report exists, open it and check the content.
  4. If the list is empty — that is OK for now; reports appear when the system generates them (or after schedule rules run).
  5. Tell us which report format you need first.
