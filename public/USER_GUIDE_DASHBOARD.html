Dashboard — user guide

Where in the app: top menu → Dashboard

Who can use it: Admin, Project Manager, and field roles (each sees a version for their work).

Current status: this section is still early. You may see a basic live summary of projects and activity. Full dashboards (charts, alerts, personal “what needs me today”) will be built when you ask for them — we will add features based on your requirements.

What this page is for

A quick “home screen” so you see important things without opening every project:

  • how projects are doing
  • urgent items and recent changes
  • messages or alerts that need attention

What you can do today

  • Open Dashboard after login
  • Look at the current summary (if data is loaded)
  • Use Refresh if the page has a refresh button

What we will add later (on request)

  • Clear KPI tiles (open tasks, overdue, active crews, etc.)
  • Alert list you can click into a project/task
  • Activity feed (who did what)
  • Different layouts for office vs field users

How to test (simple)

  1. Log in as Admin or Project Manager.
  2. Open Dashboard.
  3. Confirm the page loads without errors.
  4. Write down what you want to see first — we will implement that next.
