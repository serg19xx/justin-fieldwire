Запись на звонок через SMS

FieldWire — Project Manager предлагает клиенту три времени; клиент отвечает 1, 2 или 3; событие попадает в Calendar

Новая функция в разделе Clients: согласование времени звонка по SMS без ручной переписки. После подтверждения клиентом слота у PM автоматически создаётся событие в календаре.

Как подбираются слоты

PM сам выбирает дату (не раньше чем через 3 дня от сегодня, без ограничения «неделей»).

FieldWire показывает полное расписание дня с шагом 30 минут (9:00–17:00). Занятые окна помечены как недоступные (событие в Calendar или слот уже удержан другим SMS-инвайтом).

PM отмечает от 1 до 3 свободных слотов для клиента. После отправки они удерживаются 3 дня до ответа клиента или истечения срока.

Как это работает

PM → FieldWire → SMS клиенту (3 слота)
Клиент → ответ SMS: 1, 2 или 3
FieldWire → событие в Calendar PM + SMS-подтверждение клиенту
  • Приглашение действует 72 часа.
  • Новый инвайт тому же клиенту отменяет предыдущий неподтверждённый.
  • Ответ должен прийти с того же номера, на который отправлено SMS.
  • Часовой пояс слотов по умолчанию: America/Toronto.

Где найти в приложении

  1. Войдите как Admin или Project Manager.
  2. Clients → список (Pharmacies, Physicians, Pharmacists, Medical clinics).
  3. В строке клиента с телефоном — Schedule call via SMS.
  4. Заполните дату, три слота, длительность → Send meeting invite.
У клиента в карточке должен быть телефон (cell / office / phone в зависимости от типа списка).

Пример SMS клиенту

FieldWire — meeting request Hi Dr. Smith, Jane Doe (Project Manager) would like to schedule a call with you. Topic: Call with Dr. Smith Date: Jun 22, 2026 (30 min) Reply with your preferred time: 1) 10 AM 2) 2 PM 3) 3:30 PM Text back 1, 2, or 3.

Пример подтверждения (ответ клиента «2»)

FieldWire — confirmed Thank you, Dr. Smith! Call with Dr. Smith Jun 22, 2026 at 2 PM — Jane Doe, Project Manager

Инструкция для Project Manager

  1. Выберите клиента с телефоном → Schedule call via SMS.
  2. Проверьте тему, дату, три времени и длительность.
  3. Сверьте блок SMS preview с тем, что увидит клиент.
  4. Send meeting invite.
  5. После ответа клиента откройте Calendar — должно появиться событие.

Production-режим

SMS отправляются на реальные номера из карточек клиентов через Twilio. Тестовый режим перенаправления отключён.
