Clients: SMS, Email и Fax

FieldWire — сообщения контактам из реестра клиентов (Admin и Project Manager)

Отправка SMS, email и fax аптекам, врачам, фармацевтам и медклиникам из меню Clients — аналог сообщений в legacy EasyRx, встроенный в FieldWire.

Кто может пользоваться

РольДоступ
AdministratorПолный доступ
Project ManagerПолный доступ
Другие ролиМеню Clients и отправка недоступны

Где найти

  1. Войдите в FieldWire.
  2. Откройте Clients в главном меню.
  3. Выберите список: Pharmacies, Physicians & Providers, Pharmacists или Medical clinics.
  4. Иконки в строке (SMS, email, fax) или выделение нескольких строк для массовой SMS/email.

У фармацевтов только SMS и email (без fax).

SMS

  1. Зелёная иконка SMS в строке.
  2. Текст сообщения (счётчик символов).
  3. Send SMS.

Массово: 2+ строк → Send SMS. Без телефона — пропуск.

Email

  1. Иконка email в строке.
  2. Email format:
    • Free form letter — свой subject и текст.
    • Шаблон SendGrid — вёрстка из SendGrid; имя клиента подставляется.
  3. Send Email.

Массово: выделение строк → Send Email.

Fax

  1. Иконка fax в строке.
  2. По желанию: Subject, Cover text.
  3. Файл: PDF или изображение, макс. 50 МБ.
  4. Send Fax. Большие файлы — до ~1 минуты.
Нужны cover text или вложение (или оба). Массовый fax не поддерживается.

Фильтр контактов

Filter: Phone / Email / Fax + Has value / Empty — найти клиентов с контактом или без.

Export CSV

Export CSV — выгрузка списка. Load from CSV пока недоступен.

Как устроено технически

Браузер вызывает API FieldWire; отправка через серверные сервисы (ключи только на API, не во frontend):

КаналСервис
EmailSendGrid (API + dynamic templates)
SMSTwilio
FaxHumbleFax (обложка + файл → очередь отправки)

Fax: черновик на HumbleFax → загрузка вложения → отправка на номер из карточки клиента.

Отдельная функция: PM предлагает клиенту три времени для звонка; клиент отвечает SMS 1, 2 или 3; событие создаётся в Calendar.
