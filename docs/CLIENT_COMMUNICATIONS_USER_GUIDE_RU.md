# Руководство: SMS, Email и Fax из раздела Clients (FieldWire)

Отправка операционных сообщений аптекам, врачам, фармацевтам и медклиникам прямо из реестра **Clients**, без выхода из FieldWire.

## Как открыть это руководство

- **Веб (после деплоя):** https://fieldwire.medicalcontractor.ca/CLIENT_COMMUNICATIONS_USER_GUIDE_RU.html
- **English:** https://fieldwire.medicalcontractor.ca/CLIENT_COMMUNICATIONS_USER_GUIDE.html
- **Из приложения:** на любой странице списка Clients — ссылка **SMS, Email и Fax** в шапке страницы (новая вкладка).

## Кто может пользоваться

| Роль | Доступ |
|------|--------|
| Administrator | Да |
| Project Manager | Да |
| Остальные роли | Нет — меню Clients и кнопки отправки скрыты |

## Где найти

1. Войдите в FieldWire.
2. Откройте **Clients** в главном меню.
3. Выберите список: **Pharmacies**, **Physicians & Providers**, **Pharmacists** или **Medical clinics**.
4. Используйте иконки в строке (SMS, email, fax) или выделите несколько строк для массовой SMS/email.

## Каналы связи

| Канал | Списки | Один клиент | Массовая отправка | Примечания |
|-------|--------|-------------|-------------------|------------|
| **SMS** | Аптеки, врачи, фармацевты | Да | Да | Только текст, макс. 1600 символов |
| **Email** | Все четыре списка | Да | Да | Свободное письмо или шаблон SendGrid |
| **Fax** | Аптеки, врачи, медклиники | Да | Нет | PDF/изображение до 50 МБ + обложка |

У фармацевтов только **SMS и email** (без fax).

## SMS

1. Нажмите зелёную иконку **SMS** в строке (номер из карточки клиента).
2. Введите текст (счётчик символов; длинные тексты — несколько сегментов SMS).
3. **Send SMS**.

**Массово:** выделите 2+ строк → **Send SMS** в панели выбора. Без телефона — пропуск; в ответе видно, сколько отправлено и сколько ошибок.

## Email

1. Синяя иконка **email** в строке.
2. **Email format:**
   - **Free form letter** — свой subject и текст.
   - **Шаблон SendGrid** — из списка; тема и вёрстка из SendGrid, имя клиента подставляется автоматически.
3. **Send Email**.

**Массово:** как SMS — выделение строк → **Send Email**.

## Fax

1. Иконка fax в строке (номер fax из карточки).
2. По желанию: **Subject**, **Cover text** (титульный лист).
3. **Fax document:** PDF или изображение (JPEG, PNG, GIF, TIFF, BMP, WebP), макс. **50 МБ**.
4. **Send Fax**. Большие файлы могут обрабатываться до ~1 минуты.

Нужны **cover text или вложение** (или оба). Массовый fax не поддерживается.

## Фильтр контактов

**Filter: Phone / Email / Fax** + **Has value** / **Empty** — найти клиентов, которым можно (или нельзя) отправить сообщение.

## Export CSV

**Export CSV** — выгрузка текущего списка с учётом фильтров. **Load from CSV** пока не реализован.

## Как это устроено технически

Браузер **не** отправляет сообщения напрямую. Приложение вызывает API FieldWire; сервер проверяет роль, читает запись клиента и отправляет через внешние сервисы:

| Канал | Сервис | На сервере |
|-------|--------|------------|
| Email | SendGrid | API key, отправитель, шаблоны |
| SMS | Twilio | SID, token, исходящий номер |
| Fax | HumbleFax | API keys, caller ID, обложка + файл |

Секреты только на **API-сервере**, не во frontend. Перед отправкой UI показывает номер/адрес из карточки клиента.

## Советы

| Проблема | Что проверить |
|----------|----------------|
| Нет кнопки отправки | Роль Admin или Project Manager |
| «No phone/email/fax on file» | Заполнить карточку или фильтр контактов |
| Пустой список шаблонов email | SendGrid и активные шаблоны в том же аккаунте |
| Fax долго «Sending…» | Нормально для файлов нескольких МБ |
| Файл слишком большой | Лимит 50 МБ |
